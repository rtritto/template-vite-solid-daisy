# template-vite-solid

Template stack:
- **Vite**
- **Solid**
- **ESLint**
- **TypeScript**

## Clone
`git clone https://github.com/rtritto/template-vite-solid.git`

## Install
`yarn`

## Start
`yarn start`

OR

`yarn dev`
